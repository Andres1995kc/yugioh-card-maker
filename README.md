# 遊戲王卡製造機 | yugioh-card-maker

https://yugioh-card.linziyou.info/

## 功能

- 製作個人化的遊戲王卡
- 支援正體中文、日文（漢字的假名部分不支援）、英文
- 2019/09以前的卡片支援以密碼輸入的方式取得卡片資訊
